# باب 1: Isaac Sim اور مصنوعی ڈیٹا جنریشن

## سیکھنے کے مقاصد

- NVIDIA Isaac Sim پلیٹ فارم اور اس کی خصوصیات کو سمجھیں
- AI ٹریننگ کے لیے مصنوعی ڈیٹا جنریشن کے بارے میں سیکھیں
- یہ دریافت کریں کہ Isaac Sim وسیع تر Isaac ایکو سسٹم کے ساتھ کیسے مربوط ہوتا ہے
- روبوٹ ادراک سسٹمز کے لیے مصنوعی ڈیٹاسیٹس تخلیق کریں

## NVIDIA Isaac Sim کا تعارف

NVIDIA Isaac Sim ایک روبوٹکس سیمولیٹر ہے جو NVIDIA Omniverse پر مبنی ہے، جو خاص طور پر AI پر مبنی روبوٹکس ایپلی کیشنز کی ترقی اور ٹیسٹنگ کے لیے ڈیزائن کیا گیا ہے۔ یہ فوٹو ریئل سٹک سیمولیشن کی صلاحیتیں، ہائی فیڈیلٹی فزکس، اور Isaac روبوٹکس سافٹ ویئر سٹیک کے ساتھ بے رکاوٹ انٹیگریشن فراہم کرتا ہے۔

### Isaac Sim کی اہم خصوصیات

1. **فوٹو ریئل سٹک رینڈرنگ**: فزکلی درست رینڈرنگ کے لیے NVIDIA RTX ٹیکنالوجی کا استعمال
2. **ہائی فیڈیلٹی فزکس**: رجڈ باڈی ڈائنامکس، کولیژن اور کنٹیکٹس کی درست سیمولیشن
3. **مصنوعی ڈیٹا جنریشن**: AI ماڈلز کی ٹریننگ کے لیے لیبلڈ ڈیٹا تخلیق کرنے کے ٹولز
4. **ROS 2 انٹیگریشن**: ROS 2 کمیونیکیشن کی مقامی سپورٹ
5. **Isaac ایکسٹینشنز**: عام روبوٹکس ٹاسکس کے لیے پہلے سے بنے ہوئے ٹولز

## روبوٹکس کے لیے مصنوعی ڈیٹا

روبوٹکس AI کی ترقی میں مصنوعی ڈیٹا جنریشن انتہائی اہم ہے کیونکہ:

- حقیقی دنیا کا ڈیٹا اکٹھا کرنا مہنگا، وقت طلب اور خطرناک ہو سکتا ہے
- مصنوعی ڈیٹا کنٹرولڈ تجربات کی اجازت دیتا ہے جن میں گراؤنڈ ٹروتھ معلوم ہو
- متنوع منظرنامے سیمولیٹ کیے جا سکتے ہیں تاکہ ماڈل کی جنرلائزیشن بہتر ہو
- مضبوطی کی ٹیسٹنگ کے لیے ایج کیسز کو خاص طور پر تخلیق کیا جا سکتا ہے

### مصنوعی ڈیٹا کی اقسام

1. **RGB تصاویر**: ویژن پر مبنی ادراک کے لیے فوٹو ریئل سٹک تصاویر
2. **ڈیپتھ میپس**: 3D سمجھ کے لیے ڈیپتھ کی معلومات
3. **سیمینٹک سیگمینٹیشن**: منظر میں اشیاء کی پکسل لیول لیبلنگ
4. **انسٹنس سیگمینٹیشن**: ایک ہی کلاس کی انفرادی اشیاء کی شناخت
5. **باؤنڈنگ باکسز**: آبجیکٹ ڈیٹیکشن کے لیے 2D/3D باؤنڈنگ باکسز
6. **پوز ڈیٹا**: پوز اسٹیمیشن ٹریننگ کے لیے اشیاء کی گراؤنڈ ٹروتھ پوز

## Isaac Sim کی آرکیٹیکچر

Isaac Sim NVIDIA Omniverse پر تعمیر کیا گیا ہے، جو ایک سیمولیشن اور کولیبریشن پلیٹ فارم ہے:

- **USD (Universal Scene Description)**: بنیادی منظر کی نمائندگی کا فارمیٹ
- **Omniverse Kit**: ایپلی کیشن فریم ورک
- **PhysX**: NVIDIA کا فزکس انجن
- **RTX رینڈرر**: فوٹو ریئل سٹک رینڈرنگ کے لیے
- **ROS 2 برج**: ROS 2 مڈل ویئر کے ساتھ انٹیگریشن

## Isaac Sim کی سیٹ اپ

Isaac Sim کو کئی طریقوں سے چلایا جا سکتا ہے:

1. **ڈاکر کنٹینر**: آسان سیٹ اپ اور مستقل مزاجی کے لیے تجویز کردہ
2. **سٹینڈ الون ایپلی کیشن**: زیادہ کنٹرول اور ایڈوانسڈ فیچرز کے لیے
3. **کلاؤڈ ڈپلائی منٹ**: NVIDIA DGX Cloud یا دیگر GPU کلاؤڈ سروسز کا استعمال

### بنیادی ڈاکر سیٹ اپ

```bash
# Isaac Sim کنٹینر کو پل کریں
docker pull nvcr.io/nvidia/isaac-sim:latest

# Isaac Sim چلائیں (NVIDIA GPU اور ڈرائیورز کی ضرورت ہے)
xhost +local:docker
docker run --gpus all -it --rm \
  --network=host \
  --env="DISPLAY" \
  --env="QT_X11_NO_MITSHM=1" \
  --volume="/tmp/.X11-unix:/tmp/.X11-unix:rw" \
  --volume="/home/user/project:/project:rw" \
  --volume="/home/user/.nvidia-ml:/usr/lib/x86_64-linux-gnu:ro" \
  nvcr.io/nvidia/isaac-sim:latest
```

## مصنوعی ڈیٹا پائپ لائنز کی تخلیق

Isaac Sim Replicator فریم ورک مصنوعی ڈیٹا جنریشن کی اجازت دیتا ہے:

```python
import omni.replicator.core as rep

# ایک سادہ مصنوعی ڈیٹا پائپ لائن کی تعریف کریں
with rep.new_layer():
    # روبوٹ ایسٹ لوڈ کریں
    robot = rep.load.usd('path/to/robot.usd')

    # ایک کیمرہ تخلیق کریں
    camera = rep.get.camera('/Replicator/Render/SmartSync/Camera')

    # رینڈمائزیشن آپریشنز کی تعریف کریں
    with robot:
        rep.randomizer.placement(
            position=rep.distribution.uniform((-100, -100, 0), (100, 100, 0)),
            rotation=rep.distribution.uniform((0, 0, -1.57), (0, 0, 1.57))
        )

    # مختلف ڈیٹا اقسام کے لیے رائٹرز رجسٹر کریں
    rep.WriterRegistry.enable_writer("basic_writer")

    # ڈیٹا جنریٹ کریں
    rep.run()
```

## USD برائے منظر کی تفصیل

Universal Scene Description (USD) 3D منظرناموں کی تفصیل کے لیے ایک طاقتور فارمیٹ ہے:

- **لیئرڈ کمپوزیشن**: منظرنامے متعدد لیئرڈ فائلوں سے بنائے جا سکتے ہیں
- **ویرینٹ سیٹس**: ایک ماڈل کی مختلف کنفیگریشنز کو ایک ہی فائل میں محفوظ کیا جا سکتا ہے
- **اینیمیشن**: پیچیدہ اینیمیشنز اور رگس کی سپورٹ
- **ایکسٹینشنز**: مختلف ڈومینز کے لیے ایکسٹینشنز کا امیر ایکو سسٹم

## Isaac ROS انٹیگریشن

Isaac Sim Isaac ROS پیکیجز کے ساتھ GPU ایکسلریٹڈ ادراک کے لیے مربوط ہوتا ہے:

- **امیج پائپ لائن**: GPU ایکسلریٹڈ امیج پروسیسنگ
- **SLAM**: سیملٹینئس لوکلائزیشن اینڈ میپنگ
- **آبجیکٹ ڈیٹیکشن**: ریئل ٹائم آبجیکٹ ڈیٹیکشن
- **مینیپیولیشن**: روبوٹک مینیپیولیشن کے لیے ٹولز

## مصنوعی ڈیٹا جنریشن ورک فلو

1. **منظر کی تخلیق**: متنوع اشیاء اور لائٹنگ کے ساتھ حقیقی ماحول ڈیزائن کریں
2. **سینسر سیمولیشن**: حقیقی ہارڈویئر سے مماثل ورچوئل سینسرز کنفیگر کریں
3. **رینڈمائزیشن**: اشیاء، ٹیکسچرز، لائٹنگ اور کیمرہ پیرامیٹرز کو مختلف کریں
4. **ڈیٹا جنریشن**: لیبلڈ ڈیٹاسیٹس جنریٹ کرنے کے لیے سیمولیشن چلائیں
5. **ویلیڈیشن**: یقینی بنائیں کہ مصنوعی ڈیٹا کا معیار اور ڈسٹری بیوشن حقیقی ڈیٹا سے مماثل ہو

## خلاصہ

Isaac Sim روبوٹکس میں AI کی ترقی کے لیے ایک طاقتور پلیٹ فارم فراہم کرتا ہے، جس میں مصنوعی ڈیٹا جنریشن کی خاص طاقت ہے۔ یہ صلاحیت مضبوط روبوٹ ادراک سسٹمز کی ٹریننگ کے لیے ضروری ہے بغیر وسیع حقیقی دنیا کے ڈیٹا اکٹھا کرنے کی ضرورت کے۔
